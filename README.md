# cellular-automata
My implementations of various cellular automata

# Implementations
* Conway's Game of Life
* Langton's Ant

# How to run
Compile with Maven
```Bash
$ mvn package
```

Run the jar
```Bash
$ java -cp target/cellular-automata-1.0-SNAPSHOT.jar com.cells.App
```

To generate files for Intellij:
```Bash
$ mvn idea:idea
```

To generate files for Eclipse:
```Bash
mvn eclipse:eclipse
```
